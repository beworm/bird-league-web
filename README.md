## Environment Variables

| Variable | Description |
|----------|-------------|
| `VITE_API_URL` | Backend API URL (default: Railway production URL) |

## Deploy to Vercel

Connect this repo to Vercel. It will auto-detect Vite and deploy.

Set `VITE_API_URL` in Vercel's Environment Variables settings if your Railway URL differs from the default.
